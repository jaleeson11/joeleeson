# Joe Leeson - Portfolio

Personal portfolio website for Joe Leeson, a freelance web developer based near London, UK.

## Tech Stack

- HTML5, SCSS, JavaScript
- jQuery, Bootstrap 5
- Laravel Mix (build tooling)

## Development

```bash
# Install dependencies
npm install

# Watch for changes
npm run watch

# Build for production
npm run production
```
